A modern, responsive landing page for **Sammunat LLC** built with **Next.js**, **React**, and **Tailwind CSS**. The project showcases a clean UI/UX design with interactive components like a responsive navbar, services section, client reviews, contact form, and footer.

---


## Features

- Responsive **navbar** with mobile toggle menu
- Hero section with call-to-action button
- Services section showcasing company offerings
- "Why Choose Us" section highlighting company strengths
- Client **reviews slider** powered by Swiper.js
- Fully functional **contact form** with SweetAlert2 notifications
- Interactive Google Maps embed
- Modern, responsive **footer** with newsletter subscription
- Fully mobile-friendly with smooth transitions and hover effects

---

## Technologies Used

- **Frontend:** React 19, Next.js 16
- **Styling:** Tailwind CSS 4
- **Components & Utilities:** React Icons, Swiper, SweetAlert2
- **Build Tools & Linting:** ESLint, PostCSS
